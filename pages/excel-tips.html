🔄 피벗테이블 사용법

1단계 — 데이터 준비
데이터는 반드시 첫 행에 제목(헤더)이 있어야 합니다. 빈 행·빈 열 없이 연속된 표 형태로 만드세요. 셀 병합은 금지.
2단계 — 피벗테이블 삽입
데이터 내 아무 셀 클릭 → 삽입 탭 → 피벗테이블 → 새 워크시트 선택 → 확인
3단계 — 필드 배치
오른쪽 필드 목록에서 항목을 드래그하세요:
• 행 영역: 분류 기준 (예: 부서명, 날짜)
• 값 영역: 합계·평균을 계산할 숫자 데이터
• 열 영역: 교차 비교할 항목
• 필터 영역: 전체 데이터 필터링
4단계 — 집계 방식 변경
값 영역 항목 클릭 → 값 필드 설정 → 합계 / 평균 / 개수 / 최대·최소 선택
꿀팁: 피벗테이블 → 슬라이서 삽입하면 버튼 클릭으로 필터링 가능. 원본 데이터 변경 후 Alt + F5로 새로 고침.
