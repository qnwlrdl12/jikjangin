이메일 서명 생성 — 자주 묻는 질문

이메일 서명은 첫인상을 결정하는 명함입니다. 이름·직함·연락처·회사명을 포함한 심플한 서명이 가장 효과적입니다. HTML 서명은 대부분의 메일 클라이언트(Gmail, Outlook, 네이버 등)에서 호환되며, 로고·색상으로 브랜드 인지도를 높일 수 있습니다.

Q. HTML 서명을 Gmail에 어떻게 적용하나요?

Gmail → 설정(톱니바퀴) → 모든 설정 보기 → 일반 탭 → 서명 섹션에서 'HTML 모드'로 전환한 뒤 복사한 HTML 코드를 붙여넣으세요. 저장 후 새 메일 작성 시 자동으로 적용됩니다.

Q. Outlook에서 HTML 서명을 넣을 수 있나요?

파일 → 옵션 → 메일 → 서명 → 새 서명 추가에서 서명 편집창의 HTML 모드에서 붙여넣으세요. Outlook은 일부 CSS 스타일을 무시할 수 있으므로 단순한 스타일이 더 안정적입니다.

Q. 이메일 서명에 이미지/로고를 넣으면?

외부 링크 이미지는 수신자의 이미지 차단 설정에 따라 표시되지 않을 수 있습니다. 텍스트 기반 서명이 가장 호환성이 높고 스팸 필터에도 유리합니다. 로고는 상황에 따라 선택적으로 추가하세요.

Q. 서명에 꼭 포함해야 하는 정보는?

이름·직함·회사명·연락처(전화 또는 이메일) 4가지가 기본입니다. 고객 대상 메일이라면 회사 웹사이트도 추가하고, 내부 메일에는 간결하게 유지하는 것이 좋습니다.

※ 입력한 정보는 서버에 저장되지 않으며 브라우저에서만 처리됩니다. 개인정보 유출 걱정 없이 안전하게 사용할 수 있습니다.
