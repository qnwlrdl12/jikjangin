🎨 스타일 & 자동 목차 만들기

⚠️ 먼저 스타일을 사용해야 목차가 생성됩니다
직접 폰트 크기를 키우고 굵게 하는 방식은 ❌. 반드시 홈 탭의 스타일 갤러리에서 제목1·제목2·제목3을 클릭해 적용하세요.
1
스타일 적용: 제목으로 만들 텍스트에 커서 위치 → 홈 탭 → 제목1(대제목), 제목2(중제목), 제목3(소제목) 클릭
2
목차 위치 지정: 목차를 넣을 빈 페이지에 커서를 두세요 (보통 표지 다음 페이지)
3
자동 목차 삽입: 참조 탭 → 목차 → 자동 목차 1 또는 2 선택 → 즉시 생성
4
업데이트: 내용이 바뀌면 목차 클릭 → 목차 업데이트 → "전체 목차 업데이트" 선택
꿀팁: 스타일을 마우스 오른쪽 클릭 → 수정을 눌러 원하는 폰트/크기/색상으로 바꿀 수 있습니다. 한 번만 설정하면 같은 스타일이 적용된 모든 제목에 자동 반영됩니다.
💡 탐색 창 활용: 보기 탭 → 탐색 창 체크 → 왼쪽에 문서 구조가 나타납니다. 클릭하면 해당 섹션으로 바로 이동 — 긴 문서에서 필수입니다.

🖨️ 페이지 번호 & 머리글/바닥글

1
페이지 번호 삽입: 삽입 탭 → 페이지 번호 → 위치 선택 (아래쪽 가운데 권장)
2
표지 번호 제외: 디자인 탭 → 첫 페이지를 다르게 체크 → 표지에 번호 안 표시
3
번호 형식: 페이지 번호 더블클릭 → 페이지 번호 형식에서 로마자(i, ii) 또는 아라비아 숫자 선택
시작 번호 조정: 특정 페이지부터 1번으로 시작하려면 그 앞에 구역 나누기를 삽입하고 "이전 섹션과 연결"을 해제한 후 시작 번호를 1로 설정하세요.

✏️ 추적 변경 & 주석 (협업 필수)

추적 변경 켜기/끄기: Ctrl+Shift+E 또는 검토 탭 → 변경 내용 추적. 켜진 상태에서 편집하면 모든 수정·삭제가 색상으로 표시됩니다. 최종 승인 전까지 원본과 비교 가능.
주석 달기: 댓글을 남길 텍스트 선택 → Ctrl+Alt+M 또는 삽입 탭 → 메모. 오른쪽 여백에 메모 말풍선이 생깁니다. 응답·해결도 가능.
변경 사항 수락·거부: 검토 탭 → 수락 또는 거부로 변경 내용을 하나씩 처리. "모두 수락"으로 한 번에 확정할 수 있습니다.
⚠️ 문서 제출 전 확인: 검토 탭 → 문서 검사로 숨겨진 메모·추적 변경이 남아 있는지 꼭 확인하세요. 의도치 않게 수정 이력이 공유될 수 있습니다.
두 문서 비교: 검토 탭 → 비교 → 두 파일을 선택하면 변경 사항을 자동으로 추적 변경 형태로 보여줍니다. 여러 버전을 합칠 때도 유용합니다.

🔒 문서 보호 & 전자서명

🔐
편집 제한
검토 탭 → 편집 제한 → "이 종류의 편집만 허용"으로 서식 변경이나 내용 수정을 막을 수 있습니다.
🔑
비밀번호 암호화
파일 탭 → 정보 → 문서 보호 → 암호화. 비밀번호를 잊으면 복구 불가이므로 주의.
📧
PDF로 내보내기
파일 → 내보내기 → PDF/XPS 만들기. Ctrl+P에서 PDF 프린터 선택도 가능. 서식이 무너지지 않아 최종 납품용으로 권장.
☁️
OneDrive 공동 편집
OneDrive에 저장 후 공유하면 여러 명이 동시에 편집 가능. 변경 사항이 실시간으로 반영됩니다.
